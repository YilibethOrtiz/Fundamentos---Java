# ☕ Fundamentos de Java y Lógica de Programación

¡Bienvenida a mi espacio de aprendizaje! En este repositorio público documento mi evolución académica y profesional en el lenguaje Java. Aquí no solo guardo código, sino que organizo la base teórica y los algoritmos que voy dominando paso a paso.

---

## 🗺️ Ruta de Aprendizaje

Este repositorio está estructurado en módulos cronológicos para mantener un orden profesional y facilitar la navegación por los conceptos:

### 📁 [Módulo 1: Sintaxis Básica y Variables](./01_Sintaxis_Basica/)
*   Configuración del entorno de desarrollo (JDK y IDE).
*   Tipos de datos primitivos (`int`, `double`, `boolean`, `char`).
*   Declaración de variables y operadores aritméticos.

### 📁 [Módulo 2: Estructuras de Control](./02_Estructuras_Control/)
*   Lógica condicional: Sentencias `if`, `else if`, y `else`.
*   Toma de decisiones múltiples con `switch`.
*   Bucles e iteraciones: `while`, `do-while`, y `for`.

### 📁 [Módulo 3: Lógica y Algoritmos Aplicados](./03_Logica_Algoritmos/)
*   Resolución de problemas matemáticos y lógicos mediante código.
*   Manipulación de estructuras simples (como promedios y contadores).

### 📁 [Módulo 4: Introducción a la Programación Orientada a Objetos (POO)](./04_POO_Fundamentos/)
*   Concepto de Clases y Objetos.
*   Definición de Atributos y Métodos.
*   Instanciación y primeros pilares de POO.
